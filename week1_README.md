# Week 1: CLI To-Do List Application

## Description
This is a simple Command Line Interface (CLI) To-Do List application written in Python. It allows users to manage their daily tasks directly from the terminal. 

## Features
- **Add Task**: Users can input a task description, which is then added to a running list of tasks.
- **View Tasks**: Displays all the added tasks in a neat bulleted list format.
- **Exit**: Allows the user to gracefully close the program.
- **Input Handling**: Handles invalid menu selections by alerting the user.

## How to Run the Code
1. Open your terminal or command prompt.
2. Navigate to the folder containing `week1.py`.
3. Run the script using Python:
   ```bash
   python week1.py
   ```
