# Week 4: Student Grade Manager

## Description
This is a command-line Student Grade Manager application written in Python. It allows teachers or users to store student records (names and marks) and calculate academic statistics such as the class average and identifying the top performer.

## Features
- **Add Student**: Prompts for a student name and their marks, saving them in a running list.
- **View Students**: Lists all added student records with their corresponding marks.
- **Average Marks**: Dynamically computes the average score of all registered students.
- **Highest Scorer**: Evaluates the list of students to find and display the student with the highest marks.
- **Empty State Protection**: Displays a "No records found" message if statistics or listings are requested before any student records are added.

## How to Run the Code
1. Open your terminal or command prompt.
2. Navigate to the folder containing `week4.py`.
3. Run the script using Python:
   ```bash
   python week4.py
   ```
