## Getting Started

### Prerequisites
To run these scripts, you need Python 3 installed on your computer. You can download it from [python.org](https://www.python.org/downloads/).

### Running the Programs
To run any of the programs, open your command prompt/terminal, navigate to this project folder, and run:

- **Week 1**: `python week1.py`
- **Week 2**: `python week2.py`
- **Week 3**: `python week3.py`
- **Week 4**: `python week4.py`
